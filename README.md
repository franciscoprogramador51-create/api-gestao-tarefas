# 🚀 Task Manager API - Backend

Este é um projeto de **Gestão de Tarefas** desenvolvido para demonstrar habilidades em **Node.js** e construção de **APIs RESTful**. É o meu primeiro projeto de portfólio backend!

## 🛠️ Tecnologias Utilizadas
- **Node.js**: Ambiente de execução.
- **Express**: Framework para criação de rotas.
- **JSON Database**: Persistência de dados local (leitura e escrita de arquivos).
- **Insomnia**: Ferramenta para testes de endpoints.

## 📌 Funcionalidades (CRUD)
- [x] **POST `/tasks`**: Cria uma nova tarefa com ID único baseado em Timestamp.
- [x] **GET `/tasks`**: Lista todas as tarefas salvas no arquivo `db.json`.
- [x] **DELETE `/tasks/:id`**: Remove uma tarefa específica do banco de dados local.

## 🚀 Como Rodar o Projeto
1. Clone este repositório.
2. No terminal, instale as dependências: `npm install`.
3. Inicie o servidor: `node src/server.js`.
4. Teste os endpoints em `http://localhost:3000/tasks`.
